\begin{abstract}
This work presents a cryptographic system that integrates the Elliptic Curve
Primality Proving (ECPP) algorithm into RSA key generation in order to obtain
provable prime factors. Unlike probabilistic primality tests such as
Miller--Rabin, ECPP produces a mathematical certificate of primality, increasing
trust in high-security cryptographic applications.
\end{abstract}
