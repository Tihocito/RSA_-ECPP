# RSA_ECPP
Đây là hệ thống ECPP (Elliptic Curve Primality Proving) tích hợp với RSA để tạo khóa mật mã với nguyên tố chứng minh được. Hệ thống bao gồm:

Thư viện Python triển khai ECPP đệ quy, xây dựng chứng chỉ toán học chứng minh tính nguyên tố.

Các module xử lý đường cong elliptic, điểm trên đường cong, số học mô-đun.

Cơ chế xác minh chứng chỉ đệ quy.

Bài báo học thuật (LaTeX) mô tả tích hợp ECPP vào RSA.

Bộ kiểm thử (pytest) đảm bảo tính đúng đắn.

Mục tiêu: Thay thế kiểm tra nguyên tố xác suất (Miller–Rabin) bằng chứng minh tất định, nâng cao độ tin cậy cho hệ thống mật mã an toàn cao.
